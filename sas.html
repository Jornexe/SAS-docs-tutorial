DATA Step

The DATA step is the primary programming tool for manipulating data in SAS. It is used to read, modify, and create SAS datasets. The basic structure of a DATA step is: 

                                data <dataset_name >;
              set <input_dataset_name >;
              <statements >;
            run;
  • <dataset_name > is the name of the output dataset that you want to create or modify.
  • <input_dataset_name > is the name of the input dataset that you want to read data from.
  • <statements > are the data manipulation statements that you want to use to modify the data in the input dataset and create the output dataset.

Here's an example of a simple DATA step:

                                data mydata;
              set inputdata;
              if age >= 18 then adult = 'Y';
              else adult = 'N';
            run;
  • This DATA step creates a new dataset called "mydata".
  • The set statement specifies that the input dataset is called "inputdata".
  • The if statement creates a new variable called "adult" based on the "age" variable in the input dataset.
Library

A SAS library is a collection of SAS files that are stored in a common location. You can use the LIBNAME statement to assign a library reference to a directory or folder on your computer or network. The basic syntax of the LIBNAME statement is: 

                                libname <library_reference >"<directory >";
  • <library_reference > is the name you give to the library reference. It can be any valid SAS name.
  • <directory > is the physical location of the SAS files that you want to associate with the library.

Here's an example of how to create a library: 

                                libname mylib "/path/to/files";
  • This LIBNAME statement creates a library reference called "mylib".
  • The physical location of the SAS files associated with this library is "/path/to/files".
WHERE Statement

The WHERE statement is used to subset a dataset based on certain criteria. It works by filtering the observations in the input dataset according to a specified condition. The basic syntax of the WHERE statement is: 

                                data <output_dataset >;
              set <input_dataset >;
              where <condition >;
            run;
  • <output_dataset > is the name of the output dataset that you want to create or modify.
  • <input_dataset > is the name of the input dataset that you want to read data from.
  • <condition > is the condition that the observations in the input dataset must meet to be included in the output dataset.

Here's an example of a WHERE statement:

                                data adults;
              set mydata;
              where age >= 18;
            run;
  • This DATA step creates a new dataset called "adults".
  • The set statement specifies that the input dataset is called "mydata".
  • The where statement subsets the observations in the input dataset to only include those where the age is greater than or equal to 18.

  • + Addition operator
  • - Subtraction operator
  • * Multiplication operator
  • / Division operator
  • ** Exponentiation operator
  • & Concatenation operator (joins character strings)

  • < Less than operator
  • <= Less than or equal to operator
  • and Logical AND operator

Here's an example of how to use operators: 

                                data mydata;
              set inputdata;

            run;
  • The * operator is used to multiply the "age" variable by 12 to create a new variable called "age_in_months".

IF Statement

The IF statement is used to conditionally execute SAS code based on a specified condition. The basic syntax of the IF statement is: 

                                if <condition >then <statements >;
  • <condition > is the condition that must be true for the code to be executed.
  • <statements > are the SAS statements that are executed if the condition is true.

Here's an example of an IF statement:

                                data mydata;
              set inputdata;
              if age >= 18 then status = 'Adult';
            run;
  • The if statement creates a new variable called "status" based on the "age" variable in the input dataset.
  • If the age is greater than or equal to 18, the value of "status" is set to 'Adult'.
  • If the age is less than 18, the value of "status" is undefined.
ELSE IF Statement

The ELSE IF statement is used to specify an additional condition to check if the initial IF statement is false. The basic syntax of the ELSE IF statement is: 

                                if <condition1 >then <statements1 >;
              else if <condition2 >then <statements2 >;
              else <statements3 >;
  • <condition1 > is the first condition that must be true for the first set of statements to be executed.
  • <statements1 > are the SAS statements that are executed if the first condition is true.
  • <condition2 > is the second condition that is checked if the first condition is false.
  • <statements2 > are the SAS statements that are executed if the second condition is true.
  • <statements3 > are the SAS statements that are executed if both conditions are false.

Here's an example of an IF/ELSE IF statement: 

                                data mydata;
              set inputdata;
              if age <18 then status = 'Child';
              else if age <65 then status = 'Adult';
              else status = 'Senior';
            run;
  • The if statement creates a new variable called "status" based on the "age" variable in the input dataset.
  • If the age is less than 18, the value of "status" is set to 'Child'.
  • If the age is between 18 and 64, the value of "status" is set to 'Adult'.
  • If the age is greater than or equal to 65, the value of "status" is set to 'Senior'.
IF THEN DO Statement

The IF THEN DO statement is used to conditionally execute a block of SAS code based on a specified condition. The basic syntax of the IF THEN DO statement is: 

                                if <condition >then do;
              <statements >;
            end;
  • <condition > is the condition that must be true for the code to be executed.
  • <statements > are the SAS statements that are executed if the condition is true.
  • The do and end statements enclose the block of code that is conditionally executed.

Here's an example of an IF THEN DO statement: 

                                data mydata;
              set inputdata;
              if age >= 18 then do;
                status = 'Adult';
                income_group = 'High';
              end;
            run;
  • The if statement creates new variables called "status" and "income_group" based on the "age" variable in the input dataset.
  • If the age is greater than or equal to 18, the values of "status" and "income_group" are set to 'Adult' and 'High', respectively.
  • If the age is less than 18, the values of "status" and "income_group" are undefined.
Comments

Comments in SAS are used to document code and provide context to other programmers who may be reading the code. Comments in SAS can be either inline comments or block comments.

Here's an example of an inline comment:

                                /* This is an inline comment */
            data mydata;
              set inputdata;
              if age >= 18 then status = 'Adult'; /* Another inline comment */
            run;
  • Inline comments in SAS start with /* and end with */ .
  • Anything between the /* and */ is a comment and is ignored by SAS.
  • In the example above, the first comment is an inline comment that provides context for the code that follows.
  • The second comment is an inline comment that explains what the IF statement is doing.

Here's an example of a block comment:

                                * This is a block comment;
              * It can span multiple lines;
              data mydata;
                set inputdata;
                if age >= 18 then status = 'Adult';
              run;
              * End of block comment;
            run;
  • Block comments in SAS start with * and end with ; .
  • Anything between the * and ; is a comment and is ignored by SAS.
  • In the example above, the block comment spans multiple lines and provides context for the code that follows.
Num to Char

Numeric to character conversion is the process of converting a numeric variable to a character variable. This can be useful when you want to concatenate a numeric variable with a character variable or when you want to format a numeric variable as a character string. To convert a numeric variable to a character variable, you can use the PUT function or the PUTN function.

Here's an example of using the PUT function to convert a numeric variable to a character variable: 

                                data mydata;
              set inputdata;
              age_char = put(age, 3.);
            run;
  • The PUT function converts the "age" variable to a character variable called "age_char".
  • The second argument of the PUT function specifies the format of the output string. In this case, the output string is 3 characters wide.
  • If the "age" variable has a value of 18, the "age_char" variable will have a value of " 18".

Here's an example of using the PUTN function to convert a numeric variable to a character variable: 

                                data mydata;
              set inputdata;
              age_char = putn(age, 3.);
            run;
  • The PUTN function is similar to the PUT function, but is specifically designed for numeric to character conversions.
  • The second argument of the PUTN function specifies the width of the output string, just like the PUT function.
  • If the "age" variable has a value of 18, the "age_char" variable will have a value of "018".

To convert a character variable to a numeric variable, you can use the INPUT function.

                                data mydata;

Output

In SAS, the OUTPUT statement is used to create new observations in a SAS dataset based on specified conditions. The basic syntax of the OUTPUT statement is: 

                                if <condition >then output;
  • <condition > is the condition that must be true for a new observation to be created.
  • The OUTPUT statement creates a new observation in the output dataset if the condition is true.
  • If the condition is false, the OUTPUT statement is ignored and no new observation is created.

Here's an example of using the OUTPUT statement to create new observations: 

                                data mydata;
              set inputdata;
              if age >= 18 then output;
            run;
  • The if statement checks whether the "age" variable is greater than or equal to 18.
  • If the condition is true, a new observation is created in the output dataset.
  • If the condition is false, the OUTPUT statement is ignored and no new observation is created.
  • The resulting output dataset will only contain observations where the age is greater than or equal to 18.
Appending Datasets

In SAS, the APPEND statement is used to combine two or more datasets into a single dataset. The basic syntax of the APPEND statement is: 

                                data <output_dataset >;
          set <input_dataset1 ><input_dataset2 >...;
        run;
  • <output_dataset > is the name of the dataset that will be created.
  • <input_dataset1 > , <input_dataset2 > , etc. are the names of the datasets that will be combined.
  • The SET statement specifies the order in which the input datasets will be read.
  • The resulting output dataset will contain all the observations from all the input datasets.

Here's an example of using the APPEND statement to combine two datasets: 

                                data combined_data;
          set input_data1 input_data2;
        run;
  • The SET statement specifies that observations from both "input_data1" and "input_data2" should be included in the output dataset.
  • The resulting output dataset will contain all the observations from "input_data1" followed by all the observations from "input_data2".
Merge

The MERGE statement in SAS is used to combine two or more datasets horizontally by matching observations based on common variables. The basic syntax of the MERGE statement is: 

                                data <output_dataset >;
                merge <dataset1 ><dataset2 >... <datasetN >;
                by <common_variable(s)>;
                run;
  • <output_dataset > is the name of the merged dataset that will be created.
  • <dataset1 > through <datasetN > are the datasets that will be merged.
  • <common_variable(s)> is the variable or variables that are common between the datasets and will be used to match observations.
  • The resulting dataset will have one observation for each matching set of observations in the input datasets.

Here's an example of using the MERGE statement to merge two datasets: 

                                data mergeddata;
                merge mydata newdata;
                by id;
                run;
  • The MERGE statement merges the datasets "mydata" and "newdata" by matching observations based on the "id" variable.
  • The resulting dataset will have one observation for each matching set of observations in "mydata" and "newdata".
  • If there are observations in "mydata" or "newdata" that do not have a matching observation in the other dataset, those observations will be excluded from the resulting dataset.
Sorting Data

In SAS, the PROC SORT procedure is used to sort a dataset by one or more variables. The basic syntax of the PROC SORT procedure is: 

                                proc sort data=<input_dataset >out=<output_dataset >;
          by <variable1 ><ascending/descending ><variable2 ><ascending/descending >...;
        run;
  • <input_dataset > is the name of the dataset to be sorted.
  • <output_dataset > is the name of the sorted dataset to be created.
  • <variable1 > , <variable2 > , etc. are the names of the variables to be sorted by.
  • <ascending/descending > specifies the order in which the variables should be sorted. Use "ASCENDING" for ascending order and "DESCENDING" for descending order.
  • The BY statement specifies the order in which the variables should be sorted.
  • The resulting output dataset will be sorted by the specified variables in the specified order.

Here's an example of using the PROC SORT procedure to sort a dataset: 

                                proc sort data=mydata out=sorted_data;
          by age;
        run;
  • The BY statement specifies that the "age" variable should be sorted in ascending order.
  • The resulting output dataset will be sorted by the "age" variable in ascending order.
Text Functions

In SAS, text functions are used to manipulate character strings. Some common text functions in SAS include:

  • UPCASE() : Converts a character string to uppercase.
  • LOWCASE() : Converts a character string to lowercase.
  • PROPCASE() : Converts the first letter of each word in a character string to uppercase and the rest to lowercase.
  • SUBSTR() : Extracts a substring from a character string.

Here are some examples of using text functions in SAS: 

                                * Convert a character string to uppercase;
        data mydata;
          set inputdata;
          name_upper = upcase(name);
        run;
  
        * Convert a character string to lowercase;
        data mydata;
          set inputdata;
          name_lower = lowcase(name);
        run;
  
        * Convert the first letter of each word to uppercase and the rest to lowercase;
        data mydata;
          set inputdata;
          name_proper = propcase(name);
        run;
  
        * Extract a substring from a character string;
        data mydata;
          set inputdata;
          first_name = substr(name, 1, index(name, ' ') - 1);
          last_name = substr(name, index(name, ' ') + 1);
        run;
  • The UPCASE() function is used to convert the "name" variable to uppercase.
  • The LOWCASE() function is used to convert the "name" variable to lowercase.
  • The PROPCASE() function is used to convert the "name" variable to proper case.
  • The SUBSTR() function is used to extract the first name and last name from the "name" variable.

Removing Duplicates

In SAS, there are several ways to remove duplicate observations from a dataset:

  • NODUPEKEY : Removes duplicate observations based on the values of one or more variables.
  • NODUP (DEPRICATED) : Removes duplicate observations based on all variables in the dataset. (DEPRICATED)
  • DUPOUT : Creates a new dataset that contains only the duplicate observations.

Here are some examples of using these options in SAS: 

                                * Removes duplicate observations based on the values of the "id" variable;
        data nodupekey_data;
          set inputdata;
          nodupekey id;
        run;
  
        * Removes duplicate observations based on all variables in the dataset (DEPRICATED);
        data nodup_data;
          set inputdata;
          nodup;
        run;
  
        * Creates a new dataset that contains only the duplicate observations;
        proc sort data=inputdata out=dupes nodupkey;
        by id;
        run;
        data dupeout_data;
          set dupes;
        run;
  • The NODUPEKEY option is used to remove duplicate observations based on the values of the "id" variable.
  • The NODUP (DEPRICATED) option is used to remove duplicate observations based on all variables in the dataset. (DEPRICATED)
  • The DUPOUT option is used to create a new dataset called "dupes" that contains only the duplicate observations, and then the SET statement is used to copy those observations to a new dataset called "dupeout_data".
Visual Inspection

In SAS, the PROC PRINT procedure is commonly used to visually inspect the contents of a dataset. This procedure can display the contents of a dataset in a tabular format, and can be used to quickly verify that the data is correct and complete.

Here's the basic syntax of the PROC PRINT procedure: 

                                proc print data=<dataset_name >;
        run;
  • <dataset_name > is the name of the dataset to be displayed.
  • The resulting output will display the contents of the dataset in a tabular format.

Here are some additional options that can be used with the PROC PRINT procedure:

  • VAR : Specifies which variables to display.
  • WHERE : Filters the observations displayed based on a specified condition.
  • FORMAT : Specifies the format to use for displaying variables.
  • LABEL : Specifies the label to use for displaying variables.
  • NOOBS : Excludes the observation number column from the output.
  • NOTITLE : Excludes the title from the output.

Here's an example of using the PROC PRINT procedure with some additional options: 

                                proc print data=mydata (obs=10) noobs label;
                var age name address;
                where age >= 18;
                format age 3.;
                title "First 10 Records of mydata where age is greater than or equal to 18";
                run;
  • The (obs=10) option is used to display only the first 10 observations.

  • The DROP option is used to remove variables from a dataset.

  • The VAR statement is used to display only the "age", "name", and "address" variables.
  • The WHERE statement is used to filter the observations displayed to only those where the "age" variable is greater than or equal to 18.
  • The FORMAT statement is used to display the "age" variable with a width of 3.
  • The LABEL option is used to display the variable labels instead of the variable names.
  • The NOOBS option is used to exclude the observation number column from the output.
  • The NOTITLE option is used to exclude the title from the output.

This will display the first 10 records of the "mydata" dataset where the "age" variable is greater than or equal to 18, and will look something like this: 

                                First 10 Records of mydata where age is greater than or equal to 18
            
                Name              Age    Address
                ------------------------------------------------------
                John Smith        23     123 Main St
                Jane Doe          19     456 Oak Ave
                Bob Johnson       21     789 Maple Dr

Here, the output displays the first 10 observations in the "mydata" dataset where the "age" variable is greater than or equal to 18, and only shows the "name", "age", and "address" variables. The output also includes a title and excludes the observation number column.

Creating a New Dataset

In SAS, there are several ways to create a new dataset. Here are some of the most common methods:

  • DATALINES : Specifies the data to be included in the dataset directly in the SAS program.
  • CARDS : Specifies the data to be included in the dataset directly in the SAS program, similar to DATALINES .
  • INPUT : Specifies the format of the input data and how to parse the data into variables.
  • LENGTH : Specifies the length of each variable in the dataset.

Here are some examples of using these methods to create a new dataset: 

                                * Using DATALINES to create a new dataset called mydata;
                data mydata;
                input name $ age gender $;
                datalines;
                John 25 M
                Jane 30 F
                Bob 40 M
                ;
                run;
        
                * Using CARDS to create a new dataset called mydata;
                data mydata;
                input name $ age gender $;
                cards;
                John 25 M
                Jane 30 F
                Bob 40 M
                ;
                run;
        
                * Using INPUT to create a new dataset called mydata;
                data mydata;
                input name $ 1-10 age 11-12 gender $ 13;
                datalines;
                John      25M
                Jane      30F
                Bob       40M
                ;
                run;
        
                * Using LENGTH to create a new dataset called mydata;
                data mydata;
                length name $ 10 age 3 gender $ 1;
                input name $ age gender $;
                datalines;
                John 25 M
                Jane 30 F
                Bob 40 M
                ;
                run;
  • The first two examples use DATALINES and CARDS to specify the data to be included in the dataset directly in the SAS program.
  • The third example uses INPUT to specify the format of the input data and how to parse the data into variables.
  • The fourth example uses LENGTH to specify the length of each variable in the dataset.
  • All four examples use the INPUT statement to define the variables in the dataset.
  • The DATALINES and CARDS statements indicate the start of the input data.
  • The $ sign after the variable names "name" and "gender" indicates that these variables are character variables.

In SAS, the MIN , MAX , and SUM functions can be used to calculate the minimum, maximum, and sum of a variable in a dataset, respectively. Here's the basic syntax of each function: 

                                min(<variable >)
        max(<variable >)
        sum(<variable >)
  • <variable > is the name of the variable to be used in the function.

Here's an example of using these functions in SAS: 

                                proc means data=mydata;
          var age;
          output min=min_age max=max_age sum=sum_age;

  • The VAR statement is used to specify the "age" variable.

In SAS, the PROC MEANS procedure can be used to calculate summary statistics of variables in a dataset. The CLASS statement can be used to group the data by one or more categorical variables. Here's the basic syntax of using PROC MEANS with the CLASS statement: 

                                proc means data=<dataset_name >n mean std min max sum;

Here's an example of using the PROC MEANS procedure with the CLASS statement: 

                                proc means data=mydata n mean std min max sum;
          var age;
          class gender;
        run;
  • The PROC MEANS procedure is used to calculate summary statistics of the "age" variable in the "mydata" dataset.
  • The VAR statement is used to specify the "age" variable.
  • The CLASS statement is used to group the data by the "gender" variable.

PROC TABULATE

PROC TABULATE is a SAS procedure used for creating customized tables of summary statistics and/or counts. It is a powerful tool for data exploration, analysis, and presentation.

The PROC TABULATE procedure summarizes data in a tabular format and allows you to control the format, layout, and content of the resulting tables. You can specify one or more variables to summarize, and one or more variables to group by. You can also use PROC TABULATE to generate tables with multiple levels of aggregation, such as subtotals and grand totals.

Here is an example of PROC TABULATE code: 

                                /* Example data */
                data mydata;
                    input gender $ age height weight;
                    datalines;
                    M 32 70 180
                    F 25 65 140
                    M 45 72 210
                    F 36 63 125
                    M 29 68 170
                    F 41 67 155
                    M 50 71 200
                    F 28 64 130
                    ;
                run;
                
                /* PROC TABULATE example */
                PROC TABULATE DATA=mydata;
                    VAR height weight;
                    CLASS gender age;
                    TABLE gender, age*height*(mean weight);
                RUN;

In this example, PROC TABULATE is used to summarize the variables height and weight by the variables gender and age . The resulting table displays the means of weight for each combination of gender and age , along with the corresponding height values.
